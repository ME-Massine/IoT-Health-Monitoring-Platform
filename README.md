# IoT Health Monitoring Platform

A software platform that simulates connected medical devices and monitors patient vital signs in real time.

## Tech Stack

- Spring Boot
- PostgreSQL
- React
- Python Sensor Simulator
- WebSocket

## Project Structure

/backend  
/frontend  
/simulator  
/docs

## Branching Strategy

- main: stable version
- dev: integration branch
- feature/*: development branches

## Team Workflow

1. Create an issue
2. Create a feature branch from dev
3. Commit changes
4. Open PR into dev
5. Review and merge
